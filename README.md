# Schuldenrechner / Debt Calculator\n\nDeutsch / German & English\n\n---\n\n## English\n\n### Short description\nA small debt calculator with a graphical user interface built for Pythonista's `ui` module (iOS). Add people, record payments with one or multiple payers, choose how to split payments, and calculate resulting debts and credits.\n\n### Features\n- Enter up to 20 people.\n- Multiple payers per payment (enter amount per payer).\n- Choose split: "Everyone" or any selection of people.\n- Add payments and keep a payments list.\n- Per-person results: who they owe, who owes them, total owed, total received and net balance.\n- Input validation for names and amounts.\n\n### Requirements\n- Pythonista for iOS (provides the `ui` module). Not runnable on standard desktop Python without modification.\n\n### Installation\n1. Open Pythonista on iOS.\n2. Add the file `Schuldenrechner.py` (copy-paste or import).\n3. Run the script. The graphical interface will appear.\n\n### Usage (brief)\n1. Enter names separated by commas or spaces and click "Personen bestätigen" (Confirm people).\n2. Click "Zahler wählen" (Choose payer), select payer(s), then "Beträge eingeben" (Enter amounts) to input amounts per payer.\n3. Click "Aufteilung wählen" (Choose split) and select "Auf alle" (Everyone) or specific people.\n4. Click "Zahlung hinzufügen" (Add payment). Repeat for all payments.\n5. Click "Berechnen" (Calculate) to view per-person debts and payment history.\n\n### Known limitations\n- Multiple payer handling: the current logic divides the total amount by the number of recipients and assigns equal repayment per recipient for each payer; individual payer amounts are not proportionally distributed. This can produce incorrect balances when payers contributed different amounts.\n- No persistent storage; data is lost when the app closes.\n- `additional_receives` is present as a placeholder but unused by default.\n- Locale/currency formatting is simple euro formatting without localization.\n\n### Suggested improvements\n- Pro rata splitting for multiple payers (handle differing payer amounts correctly).\n- Save/load sessions (JSON).\n- Export results (CSV).\n- Add settlement algorithm to minimize transfers.\n- Port to desktop GUI frameworks (tkinter/Qt) for wider compatibility.\n\n---\n\n## Deutsch\n\n### Kurzbeschreibung\nEin einfacher Schuldenrechner mit grafischer Oberfläche für Pythonistas `ui`-Modul (iOS). Nicht direkt auf Standard-Desktop-Python lauffähig ohne Anpassungen.\n\n### Installation\n1. Pythonista öffnen.\n2. Datei `Schuldenrechner.py` hinzufügen (kopieren/einfügen).\n\3.
